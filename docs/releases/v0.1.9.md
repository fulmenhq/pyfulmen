## Usage Examples

### Schema Export API

```python
from pyfulmen.schema import export_schema

# Export a schema to JSON
schema_data = export_schema("foundry/v1.0.0/mime-types", format="json")

# Export with YAML format
schema_yaml = export_schema("config/logging/v1.0.0/policy", format="yaml")
```

### CLI Usage

```bash
# Export schema to JSON file
pyfulmen schema export foundry/v1.0.0/mime-types --format json --output mime-types.json

# Export to YAML with provenance
pyfulmen schema export config/logging/v1.0.0/policy --format yaml --include-provenance
```
